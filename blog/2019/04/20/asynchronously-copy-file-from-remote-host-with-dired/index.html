Asynchronously copy file from remote host with dired

Today I had to copy a large file from a remote machine. The copying process is synchronous and blocked emacs.

The solution was to use the emacs-async package. This is already installed with spacemacs, type SPC h P <async> to confirm.

To make dired copy asynchronously, put the following into your config: 

(autoload 'dired-async-mode "dired-async.el" nil t)
(dired-async-mode 1)
